5. Java的注释，标识符、标识符的命名规范

	标识符命名规则：
	
		1.标识符由26个英文字符大小写（a~zA~Z）、数字(0~9)、下划线(_)和美元符号($)组成。
		2.不能以数字开头
		3.严格区分大小写
		4.标识符的可以为任意长度
		
	命名习惯（提高代码阅读性）：
	
		包名：一般全都小写(package.xx.xxxx)
		类名，接口：多个单词组成时首字母大写（HelloWorld）
		变量名，函数名：多个单词组成时，首字母小写，其他首字母大写（getTime）
		常量名：全部大写，单词之间用_连接。
		
	注释：
	
		单行注释：//
		多行注释：/* */
		文档注释：/** */
		
		注意：文档注释只能出现在类、属性、方法的上面。
