1. Hbase的访问方式

	https://blog.csdn.net/qq_806913882/article/details/53556371
	
	1、Native Java API：最常规和高效的访问方式； 
	2、HBase Shell：HBase的命令行工具，最简单的接口，适合HBase管理使用； 
	3、Thrift Gateway：利用Thrift序列化技术，支持C++，PHP，Python等多种语言，适合其他异构系统在线访问HBase表数据； 
	4、REST Gateway：支持REST 风格的Http API访问HBase, 解除了语言限制； 
	5、MapReduce：直接使用MapReduce作业处理Hbase数据； 
	6、使用Pig/hive处理Hbase数据。

	7、spring中 java 利用 sping-data-hadoop HbaseTemplate 操作 hbase
