		引用数据类型
		
			在Java中，引用类型的变量非常类似于C/C++的指针。引用类型指向一个对象，指向对象的变量是引用变量。
			
			对象、数组都是引用数据类型。
			
			所有引用类型的默认值都是null。
			
			一个引用变量可以用来引用任何与之兼容的类型。
			
				例子：Site site = new Site("Runoob")。
			
			
	Java 常量
	
		在 Java 中使用 final 关键字来修饰常量
		
			注: 虽然常量名也可以用小写，但为了便于识别，通常使用大写字母表示常量。 
			
			final double PI = 3.1415927;
