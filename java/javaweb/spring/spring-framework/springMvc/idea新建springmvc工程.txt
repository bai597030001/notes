一、通过IDEA新建Spring MVC项目

方法一: 使用spring搭建spring mvc环境

	https://www.cnblogs.com/wormday/p/8435617.html

	1.新建springmvc项目

		new project -> spring: (勾选 spring MVC(4.3.14.RELEASE) 和 Web Application)
		
		-> project name & project location -> finish; idea会帮助下载一部分jar包
		
	2.设置运行、调试相关配置

		2.1 配置 tomcat
		
		2.2 配置 Artifact
		
	3.导入Spring MVC相关类库

	4.编写代码,修改配置

	5.springmvc使用maven管理

		5.1 右键项目 -> Add Framework Suppurt -> Maven
		5.2 新增maven项目依赖
		5.3 右键pom.xml文件选择maven->reimport注入依赖
		
		注意: 提示找不到资源，我们查看了文件路径，/WEB-INF/views/下面却有index.jsp文件，这是为何？
			原来是我们之前添加maven框架的时候更改了webapp的授权目录，点击查看根目录的iml文件springmvc.iml
			我们发现webroots路径是src/main/webapp/，但是我们项目上并没有这个目录，是故报错，
			所以这里有两个办法，一：是在对于目录创建相应的文件夹，二：是直接修改他的路径。
			这里我选用第二种，建路径改为web

方法二:	使用maven搭建spring mvc环境

	1. 新建项目
	
		File->NEW->Project选择maven，然后勾选Create form archetype 选中webapp
		
	2. 创建项目目录结构
	
	3. 配置配置文件
	
		同第一种方法一样配置web.xml，spring-mvc.xml文件，添加tomcat服务器，
		创建IndexController控制器，index.jsp视图文件之后就完成了springmvc环境搭建
		
		启动项目就可以得到如上的结果。（注该方法无需修改iml文件里面的路径，
		而web.xml文件放在/webaap/WEB-INF/路径下）
